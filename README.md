# CrackHash

**CrackHash** — это распределённая система перебора (brute-force), предназначенная для восстановления строки по заданному MD5-хэшу.

Проект демонстрирует построение масштабируемой архитектуры с использованием очередей задач и воркеров для параллельной обработки.


## Возможности

- Подбор строки по MD5-хэшу методом brute-force  
- Распределённая обработка задач через очередь  
- Горизонтальное масштабирование воркеров  
- REST API для управления задачами  
- Хранение результатов и состояния задач  

## Архитектура

Система состоит из следующих компонентов:

- **Backend API** (Django + DRF)  
  Отвечает за приём запросов, создание задач и выдачу результатов  

- **Message Broker** (RabbitMQ)  
  Управляет очередью задач  

- **Workers** (Celery)  
  Выполняют brute-force перебор  

- **Database** (MongoDB с Django MongoDB Backend)  
  Хранит задачи, статусы и результаты
